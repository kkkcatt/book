Program features:

  1.Reads a text file (e.g., a book)

  2.Counts the total number of words

  3.Counts the occurrences of each letter (A-Z or a-z)



程序功能：

  1.输入一本书的文本文件

  2.统计文件的总字数

  3.统计每个字母（A-Z 或 a-z）在文本中出现的次数
